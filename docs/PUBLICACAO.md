## Texto pronto para postar com o video

Finalizei o FarmaSys Pro, um sistema desktop para gestao operacional de farmacia desenvolvido com Java Swing e SQLite.

O projeto foi pensado para simular um ambiente real de operacao, com modulos de:

- login por perfil
- caixa / PDV
- caixa de pagamento com comanda
- controle de estoque
- recebimento de XML de nota fiscal
- historico com receita
- relatorios diarios por loja e por usuario

Foi um projeto importante para consolidar regras de negocio, organizacao de interface desktop e experiencia de uso voltada para rotina real de farmacia.
