# Textos prontos para LinkedIn e curriculo

## Versao curta

Desenvolvi o FarmaSys Pro, um sistema desktop para gestao de farmacia com Java Swing e SQLite. O projeto inclui login por perfis, PDV, caixa de pagamento com comanda, controle de estoque, recebimento por XML, historico com receita, controle de medicamentos sujeitos a prescricao e relatorios diarios por usuario e por loja.

## Versao para LinkedIn

Desenvolvi um sistema desktop completo para gestao operacional de farmacia usando Java Swing, SQLite e Maven. O projeto simula rotinas reais de balcao e administracao, com modulos de login por perfil, PDV, caixa de pagamento, controle de estoque, recebimento de XML de nota fiscal, historico com receita, cadastro de medicos e relatorios diarios.

Destaques do projeto:

- Caixa / PDV com busca inteligente de clientes e medicamentos
- Fluxo de comanda para separar venda e pagamento
- Entrada de mercadorias por XML
- Controle de medicamentos controlados e antibioticos
- Relatorios diarios por loja e por usuario
- Interface desktop com foco em uso operacional

## Versao para curriculo

### Projeto autoral - FarmaSys Pro

Sistema desktop para gestao operacional de farmacia, desenvolvido com Java Swing, SQLite e Maven.

- Implementei fluxo de venda com PDV, carrinho, comanda e caixa de pagamento
- Modelei regras de negocio para controlados, antibioticos, entrega e historico com receita
- Desenvolvi modulo de estoque com busca inteligente, ajuste manual e recebimento de mercadorias via XML
- Criei relatorios diarios com visao por loja, usuario e entregas
- Estruturei o projeto com camada DAO, modelos, utilitarios e interfaces Swing

## Versao tecnica para entrevista

Esse projeto foi importante para eu demonstrar capacidade de construir um produto de escritorio completo, com regras de negocio reais e integracao entre varios fluxos. Nao foi apenas um CRUD. Eu trabalhei com separacao por perfil, persistencia local, leitura de XML, calculo de comissao, controle de estoque, validacao de receita e uma interface pensada para o uso operacional do dia a dia.
