# Portfolio do projeto

## Nome do projeto

FarmaSys Pro

## Frase curta

Sistema desktop para gestao operacional de farmacia com PDV, caixa, estoque, XML e controle de receita.

## Problema que o projeto resolve

Farmacias trabalham com processos que exigem velocidade no atendimento, controle de estoque, registro de receitas, entrada de mercadorias por nota fiscal e acompanhamento diario das vendas. O projeto foi desenvolvido para centralizar essas rotinas em um sistema unico, com interface desktop e foco no uso real da operacao.

## Publico-alvo

- Atendentes de farmacia
- Operadores de caixa
- Chefes de loja
- Administradores

## Modulos principais

### 1. Login e perfis

Cada usuario entra com um perfil especifico e visualiza apenas o que faz sentido para sua rotina.

### 2. Caixa / PDV

Permite localizar cliente, buscar medicamentos, montar o carrinho, aplicar regras de negocio e gerar a comanda da venda.

### 3. Caixa de pagamento

Separa o fechamento financeiro da venda, permitindo conferencia dos itens e registro da forma de pagamento em uma tela dedicada.

### 4. Controle de estoque

Organiza centenas de itens com busca inteligente, dados operacionais e ajuste rapido de saldo.

### 5. Recebimento por XML

Importa a nota fiscal da distribuidora, confere os itens e atualiza o estoque com lote, validade e custo.

### 6. Relatorios e historico

Exibe faturamento diario, vendas por usuario, entregas e historico com receita.

### 7. Receita e controlados

Adiciona uma camada de rastreabilidade para medicamentos que exigem prescricao, com dados do medico, paciente e receita.

## Diferenciais

- Fluxo de comanda entre venda e pagamento
- Tratamento especial para medicamentos controlados e antibioticos
- Busca inteligente em clientes, medicos, produtos e distribuidoras
- Recebimento de estoque por XML
- Interface visual voltada para operacao real
- Catalogo grande de medicamentos e perfumaria
